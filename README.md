# cs377uchat

Files adapted from: https://snack.expo.io/@bacon/firebase-basic-chat-video-tutorial and https://www.youtube.com/watch?v=qo7xtBCh18o. 
This is to demo that GiftedChat works with Firebase and React Navigation 5. User authentication is set to anonymous. 

## To run

In the terminal: 
```npm install```

```npm install --save firebase```

```firebase login```

```firebase init```
* Select database and functions

```firebase deploy```

Then start up the expo app:
```expo start```
